# API Testing Framework (Python)

## Overview
This project is a simple API testing framework built using Python. It validates API responses, status codes, and JSON data while handling failures and logging results.

## Features
- Automated API test execution
- Status code validation
- JSON response validation
- Failure handling (continues execution)
- Logging with timestamps

## Tech Stack
- Python
- Requests library

## How to Run
pip install -r requirements.txt
python main.py

## Example Output
[PASS] test_github_api_status
[PASS] test_github_api_content
[FAIL] test_invalid_endpoint

- API Testing
- Debugging
